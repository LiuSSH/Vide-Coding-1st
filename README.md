# 俄罗斯方块

一个经典的HTML5俄罗斯方块游戏。

## 功能特点

- 完整的俄罗斯方块游戏机制
- 7种不同形状的方块，每种都有独特的颜色
- 分数和等级系统
- 下一个方块预览
- 随着分数增加，游戏速度会提升
- 游戏结束检测和重新开始功能

## 如何开始

直接在浏览器中打开 `tetris.html` 文件即可开始游戏。

## 操作说明

- **↑ 上方向键**：旋转方块
- **← 左方向键**：向左移动
- **→ 右方向键**：向右移动
- **↓ 下方向键**：加速下落
- **空格键**：游戏结束后重新开始

## 游戏规则

- 移动和旋转方块，使其填满整行
- 当一行被填满时，该行会被消除并得分
- 同时消除多行可获得更高分数
- 分数每增加1000分，等级提升，游戏速度加快
- 当方块堆积到顶部时，游戏结束

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
